I was thrilled to work on a pilot project with the city of Raleigh and NVIDIA, to explore how real-time
video analytics can be used to monitor traffic and congestion in the city. We set up a dashboard that shows 
the traffic conditions in real-time, as well as historical data on traffic patterns and congestion.
I had a lot of fun building the dashboard using ArcGIS Maps SDK for JavaScript. We got the data from 
NVIDIA's DeepStream backend and we used Velocity to ingest it and display it on the map. 
You can read more about it on NVIDIA's blog: [How Raleigh Enhances City Safery and Mobility with Physical AI](https://www.nvidia.com/en-us/case-studies/raleigh-smart-city/).
