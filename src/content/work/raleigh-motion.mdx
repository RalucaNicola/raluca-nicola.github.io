In 2025, we had a pilot project with the city of Raleigh and NVIDIA, to explore how real-time
video analytics can be used to monitor traffic and congestion in the city. We set up a dashboard that shows 
the traffic conditions in real-time, as well as historical data on traffic patterns and congestion.
I had a lot of fun working on building the dashboard using ArcGIS Maps SDK for JavaScript. We got the data from 
NVIDIA's DeepStream backend and we used Velocity to ingest it and display it on the map. 
More about this project on NVIDIA's blog: [How Raleigh Enhances City Safery and Mobility with Physical AI](https://www.nvidia.com/en-us/case-studies/raleigh-smart-city/).
And this year Raleigh was awarded the [IDC Smart Cities North America Award](https://raleighnc.gov/apps-maps-and-open-data/news/city-wins-2026-idc-smart-cities-north-america-award) for this project.
